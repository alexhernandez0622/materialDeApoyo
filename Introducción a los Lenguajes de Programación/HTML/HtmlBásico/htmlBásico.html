Introducción a HTML

Estructura básica de un documento HTML

Un documento HTML comienza con la declaración <!DOCTYPE html> que indica al navegador el tipo de documento. Luego, el contenido está estructurado con etiquetas <html>, <head> y <body>.

Etiquetas esenciales

  • <html>: La etiqueta raíz del documento que contiene todo el contenido.
  • <head>: Contiene metadatos, como el título del documento y enlaces a recursos externos.
  • <body>: Contiene el contenido visible del documento, como texto, imágenes y enlaces.
  • <h1> a <h6>: Encabezados de diferentes niveles para organizar el contenido.
  • <p>: Etiqueta de párrafo para bloques de texto.
  • <a>: Etiqueta de ancla para enlaces.
  • <img>: Etiqueta para insertar imágenes.
  • <ul> y <ol>: Etiquetas para listas desordenadas y ordenadas, respectivamente.
  • <li>: Etiqueta para elementos de lista.
  • <table>: Etiqueta para tablas.
  • <tr>, <th>, <td>: Etiquetas para filas, encabezados de columna y celdas de tabla.

Imágenes en HTML

Para insertar imágenes, se utiliza la etiqueta <img> con atributos como src (fuente) y alt (texto alternativo).

Tablas en HTML

Las tablas se crean con la etiqueta <table> y se organizan en filas (<tr>), encabezados de columna (<th>), y celdas de datos (<td>).
